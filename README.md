# TaskMaster: Efficient To-Do Manager

## Overview

**TaskMaster** is a simple yet powerful Android application designed to enhance productivity by helping users organize their tasks efficiently. The app provides a user-friendly interface that allows individuals and teams to create, store, and manage a list of tasks. With TaskMaster, users can easily track their progress and complete tasks, ensuring they stay focused and achieve their goals.

## Features

- **Create Tasks**: Users can add new tasks with a title and timestamp.
- **View Tasks**: A clear and organized list of tasks is displayed, allowing users to see what needs to be done at a glance.
- **Delete Tasks**: Once a task is completed, users can easily remove it from the list, keeping their task list clean and manageable.
- **User-Friendly Interface**: The app is designed with simplicity in mind, making it easy for anyone to use, regardless of their technical expertise.
- **Data Persistence**: Tasks are stored in a local database, ensuring that users' data is saved even when the app is closed.

## Technologies Used

- **Kotlin**: The primary programming language for Android development.
- **Android Jetpack**: Utilized for building robust Android applications.
- **Room Database**: For local data storage and management of tasks.
- **Android Compose**: For building the UI in a declarative manner.
- **LiveData**: To observe data changes and update the UI accordingly.

## Getting Started

To get started with TaskMaster, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/smq2021/TaskMaster-Efficient-To-Do-Manager.git
   ```

2. **Open the Project**:
   Open the project in Android Studio.

3. **Build the Project**:
   Ensure that you have the necessary SDKs and dependencies installed. Build the project to download any required libraries.

4. **Run the App**:
   Connect an Android device or use an emulator to run the app.

## Usage

1. **Adding a Task**:
   - Open the app and enter the title of your task in the input field.
   - Click the "Add" button to save the task.

2. **Viewing Tasks**:
   - All your tasks will be displayed in a list format.
   - You can scroll through the list to view all tasks.

3. **Deleting a Task**:
   - Once you complete a task, click the delete icon next to the task to remove it from the list.
